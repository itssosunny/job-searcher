# 잡써쳐 5분 퀵스타트

Claude Code가 처음이어도 됩니다. 아래 순서대로만 하면 5분 안에 첫 공고 수집까지 갑니다.

## 준비물

- **Claude Code** — [설치 안내](https://claude.com/claude-code). 터미널 CLI나
  데스크톱 앱(Mac/Windows) 아무거나 됩니다.
- **Python 3** — Mac은 기본 내장. 없어도 일단 진행하세요(아래 FAQ 참고).

## 3단계

**1. 플러그인 설치** — Claude Code 안에서 두 줄:

```
/plugin marketplace add itssosunny/job-searcher
/plugin install job-searcher
```

**2. 한국어로 시키기** — 그냥 말하면 됩니다:

```
잡코리아에서 데이터 엔지니어 공고 20개 모아줘
```

**3. 결과 받기** — 공고가 정규화된 JSON으로 수집됩니다. 표로 정리해달라고 하거나,
여러 플랫폼을 한 번에 시켜도 됩니다:

```
원티드, 점핏, 사람인에서 프로덕트 매니저 공고 모아서 마감일 순으로 표로 정리해줘
```

## 자주 묻는 것

**Python 패키지를 따로 설치해야 하나요?**
아니요. 수집기가 필요로 하는 패키지(`curl_cffi`, `beautifulsoup4`, `lxml`)가 없으면
에이전트가 알아서 설치하고 재시도합니다.

**로그인이 필요한가요?**
아니요. 22개 수집기 전부 공개 페이지만 읽습니다. 어떤 계정도, 어떤 로그인도 쓰지 않습니다.

**특정 플랫폼 하나만 쓰고 싶은데요.**
`skills/<플랫폼>-collect/` 디렉토리 하나만 복사해 가도 그대로 돌아갑니다.
각 스킬의 SKILL.md에 사용법이 있습니다.

**공고가 안 나와요.**
사이트가 개편되면 해당 수집기가 멈출 수 있습니다.
[이슈](https://github.com/itssosunny/job-searcher/issues)로 알려주세요 —
스킬 하나가 파일 하나라 고치기 쉽습니다.
